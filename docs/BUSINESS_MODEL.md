# Business Model Hypothesis (v1)

最終更新: 2026-07-02

## 1. ターゲット

- 学生: 学習継続・受験/資格準備
- 社会人: 副業・スキルアップ・健康習慣

## 2. 提供価値

- 未来の自己像を日次行動に接続する対話設計
- 朝WOOP + 夜振り返りの習慣ループ
- 見積もり/調整/実績データの蓄積による個別最適化（v2）

## 3. 収益仮説

1. フリーミアム
   - Free: 基本対話と履歴閲覧（低頻度）
   - Pro: 対話回数増加、週次レビュー、通知、自動最適化
2. 学生向け割引プラン
3. 将来的にB2B（学習塾・研修用途）提供

## 4. KPI

- Activation: オンボ完了率
- Habit: 7日継続率 / 朝夜両方実施率
- Value: 目標達成率（調整後）推移
- Retention: D7 / D30
- Revenue: Free→Pro転換率

## 5. 実験優先順位

1. Onboardingの完了率改善（対話→自動抽出）
2. リテンション改善（ストリーク + リマインド）
3. スコア納得感改善（単位混在除去）
4. Pro機能の価値検証（通知・週次レビュー）
